# Job Application Tracker

A web application for tracking job applications, application statuses, interviews, and wishlist jobs.

---

# Features

## Current Features
- Dashboard Overview page
- Navigation bar
- Summary cards
- Analytics section placeholders
- Job Application form
- Status dropdown selection
- Notes section
- Responsive layout using Flexbox
- Modern dashboard UI styling

---

# Planned Features
- Add applications dynamically with JavaScript
- Store applications in PostgreSQL database
- Edit/Delete applications
- Application status tracking
- Charts and analytics
- Wishlist tracking
- Notifications/reminders
- File upload for CVs
- Authentication/Login system
